## 🚀 Commands Reference

### Setup
```bash
npm install          # Install dependencies
npm start           # Start development
npm test            # Run tests
```

### Running on Devices
```bash
npm run android     # Android emulator
npm run ios        # iOS simulator
npm run web        # Web browser
```

### Production
```bash
expo build:android  # Build APK
expo build:ios     # Build IPA
```

## 🔧 Customization Guide

### Change Colors
File: `src/constants/theme.js`
```javascript
primary: '#FF5722',    // Your color
success: '#4CAF50',    // Your color
danger: '#F44336'      // Your color
```

### Add New Screen
1. Create: `src/screens/NewScreen.js`
2. Update: `src/navigation/RootNavigator.js`
3. Add navigation action

### Modify Component
1. Find component in: `src/components/`
2. Review code in: COMPONENT_ARCHITECTURE_GUIDE.md
3. Make changes
4. Test on device

## ⚠️ Common Issues

| Issue | Solution |
|-------|----------|
| Can't connect to API | Start backend: `cd BACKEND && npm run dev` |
| Camera not working | Grant permission in phone settings |
| Barcode not detected | Hold 10-20cm away, good lighting |
| npm install fails | Delete node_modules, try again |
| App crashes | Check console, backend logs |
